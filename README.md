# Node Boilerplate
My open-sourced node boilerplate that has goodies and helpers baked in - a starter template for all my Node builds.

Includes:
- Axios
- Babel
- Connect Mongo
- ES6 Promisify
- Express
- HTML to Text
- Jimp
- Juice
- MD5
- Mongoose
- Multer
- NodeMailer
- NOW
- Passport
- PostCSS / SASS
- Slugs
- UUID
- Webpack
